Line-Tracing Robot – Analog Motor Control System

Fall 2025 · Analog Systems Project

▶

As an analog systems project, a group of my classmates and I designed and implemented a line-tracing robot using continuous-time motor control. Infrared sensors were used to detect track position and generate an error signal, which was processed through analog control circuitry to dynamically adjust motor speed and direction. The system demonstrates closed-loop feedback, stability, and real-time response, highlighting how classical control techniques can be applied to autonomous robotic navigation without the use of a microcontroller.
