Projects

Analog Line-Following Robot – Multi-Phase Design

▶

Designed an analog line-tracing robot built using photodiodes/phototransistors, op-amp comparators, motor drivers, and a multi-stage control system. The robot detects a black electrical tape track, adjusts steering through the differential motor control, and stabilizes direction in real time without microcontrollers.

PID-Controlled Cruise Control Simulator – Analog System

▶

Built an analog cruise control simulator using op-amps to implement proportional, integral, and derivative control blocks. The system dynamically stabilizes vehicle speed, compensates for “hills” and disturbances, and demonstrates classical control theory in hardware.

Zener Diode Voltage Regulator – 5V Solar Adapter PCB

▶

Designed a voltage-regulated PCB using the 1N5230 Zener diode to regulate 6–12V solar input down to 5V with a BS170 MOSFET current limiter. Completed schematic simulation, board layout, PCB fabrication, load testing, and oscilloscope-based validation of regulation performance.

Digital Thermometer – LPC1768 + MCP9701

▶

Built a digital thermometer using an MCP9701 analog temperature sensor and an LPC1768 microcontroller. Implemented ADC sampling, scaling conversions, Celsius/Fahrenheit computation, and multi-LED temperature range indication. Included calibration, filtering, and user interface logic.

Brightness Controller – SPI, I²C, PWM, 7-Segment Display

▶

Developed a full embedded dimmer system featuring SPI output via a 74HC595 shift register, SPI input via a 74HC165 for button reading, I²C EEPROM state storage, and PWM-based LED brightness control. Included custom 7-segment display encoding and persistent brightness memory.
